# 文档路径图

本目录只放给人类阅读的说明、百科、研究资料、计划和历史记录。当前翻译流程的 Agent 契约以 `skills/att-mz/` 和 `skills/att-mz-release/` 为准；`docs/` 不覆盖 Skill、README、CLI、发布脚本和测试事实。

## 当前入口

| 路径 | 内容 |
| --- | --- |
| [快速开始](../README.md) | 面向发行包用户和首次使用者。 |
| [guides/](guides/README.md) | 使用教学、源码运行和源码编译说明。 |
| [wiki/](wiki/README.md) | 当前实现百科，包括数据库结构和源码模块地图。 |
| [history/project-memory.md](history/project-memory.md) | 从过程文档中浓缩出的项目历史记忆。 |

## 资料分层

| 路径 | 内容 |
| --- | --- |
| [research/](research/README.md) | 调研和设计资料，记录尚未完全落成当前实现的背景分析。 |
| [plans/](plans/README.md) | 当前开发计划入口；已完成或下线计划压缩进历史记忆。 |
| [history/](history/project-memory.md) | 浓缩历史记忆，只保留过程文档沉淀出的长期结论。 |

## 边界

百科文档描述当前实现；历史记忆只记录长期教训；计划只记录仍需推进的目标、边界和顺序。如果历史记忆或计划内容与当前代码、Skill、README 或发布脚本冲突，以当前实现和公开契约为准。
